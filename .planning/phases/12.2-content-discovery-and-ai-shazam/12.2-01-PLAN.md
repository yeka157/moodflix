**Create `lib/tmdb-cache.ts`** with:

1. TTL constants:
   - CATEGORY: 24 hours (24 * 60 * 60 * 1000)
   - RATINGS: 7 days (7 * 24 * 60 * 60 * 1000)
   - DETAILS: Infinity (permanent)

2. `isStale(updatedAt: Date | null, ttlMs: number): boolean` helper

3. Generic cache reader: `getCachedCategory(key: string, category: string, mediaType: "movie" | "tv", fetcher: () => Promise<T>): Promise<T>` -- checks tmdbCache by key, returns if fresh, else calls fetcher, upserts result fire-and-forget, returns fresh data.

4. Cached wrapper functions (same return types as originals):
   - `getCachedTrending(page?)` wraps `getTrendingMovies`
   - `getCachedPopular(page?)` wraps `getPopularMovies`

   - `getCachedOnTheAirTV(page?)` wraps `getOnTheAirTV`

   - `getCachedKoreanDramas(page?)` wraps `discoverKoreanDramas`
   - `getCachedChineseDramas(page?)` wraps `discoverChineseDramas`
