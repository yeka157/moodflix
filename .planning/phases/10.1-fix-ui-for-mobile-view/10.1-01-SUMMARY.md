## Decisions Made

- **Hide MockupFrame on mobile (not scale it):** The mockup is decorative — on mobile with flex-col layout it pushes CTAs below the sticky viewport. Hiding it at `< lg` ensures all text content (badge, headline, subline, CTAs) fits within `h-screen`.
- **Applied changes to both StaticHero and ScrollPinnedHero:** Both code paths render the hero (StaticHero for reduced-motion users, ScrollPinnedHero for animated users). Both needed the same responsive fixes to stay in sync.
- **MarqueeFallback row split:** Using `Math.ceil(length / 2)` to split posters into first/second half. Fallback to `[...posters].reverse()` for secondHalf when fewer than 4 posters ensures the second row still has visually distinct content.
