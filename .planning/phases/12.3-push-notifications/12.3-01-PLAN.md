1. Install web-push: `npm install web-push && npm install -D @types/web-push`

2. Add two tables to `drizzle/schema.ts` (after the existing `topHundred` table):

   **pushSubscriptions** table:
   - `id`: uuid, primaryKey, defaultRandom
   - `userId`: uuid, notNull, references profiles.id with onDelete cascade
   - `endpoint`: text, notNull, unique (Web Push service URL, unique per device)
   - `p256dh`: text, notNull (public encryption key)
   - `auth`: text, notNull (auth secret)
   - `createdAt`: timestamp with timezone, defaultNow

   **notificationSubscriptions** table:
   - `id`: uuid, primaryKey, defaultRandom
   - `userId`: uuid, notNull, references profiles.id with onDelete cascade
   - `tmdbId`: integer, notNull
   - `title`: text, notNull (movie title for notification body)
   - `posterPath`: text (nullable, for potential future use)
   - `releaseDate`: text (nullable, cached at subscribe time)
   - `lastNotifiedAt`: timestamp with timezone (nullable, idempotency guard per Pitfall 6)
   - `createdAt`: timestamp with timezone, defaultNow
   - Unique constraint: `notification_subs_user_tmdb_unique` on (userId, tmdbId)

1. Edit `app/sw.ts` -- add push and notificationclick event listeners BEFORE the `serwist.addEventListeners()` call:

   **Push event handler:**
   ```typescript
   self.addEventListener("push", (event) => {
     const data = event.data?.json() ?? {};
     const title = data.title ?? "Moodflix";
     const options: NotificationOptions = {
       body: data.body ?? "",
       icon: "/icons/icon-192x192.png",
       badge: "/icons/icon-72x72.png",
       data: { url: data.url ?? "/" },
     };
     event.waitUntil(self.registration.showNotification(title, options));
   });
   ```

   **Notificationclick handler:**
   ```typescript
   self.addEventListener("notificationclick", (event) => {
     event.notification.close();
     const url = event.notification.data?.url ?? "/";
     event.waitUntil(
       clients.matchAll({ type: "window" }).then((clientList) => {
         for (const client of clientList) {
           if (client.url.includes(url) && "focus" in client) {
             return client.focus();
           }
         }
         return clients.openWindow(url);
       })
     );
   });
   ```

   Place these two listeners after the `const serwist = new Serwist({...})` block but before `serwist.addEventListeners()`. Add a comment separator: `// --- Push notification handlers ---`

2. Create `vercel.json` in project root:
   ```json
   {
     "crons": [
       {
         "path": "/api/cron/release-notifications",
         "schedule": "0 8 * * *"
       }
     ]
   }
   ```

   This configures Vercel to trigger the cron route daily at 8 AM UTC.
