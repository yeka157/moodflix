## User Setup Required

Environment variables needed (from plan frontmatter):
- `NEXT_PUBLIC_VAPID_PUBLIC_KEY` - Run `npx web-push generate-vapid-keys` locally, copy public key
- `VAPID_PRIVATE_KEY` - Run `npx web-push generate-vapid-keys` locally, copy private key
- `VAPID_SUBJECT` - Set to `mailto:your-email@example.com`
- `CRON_SECRET` - Generate with `openssl rand -base64 32`, add to Vercel dashboard env vars
