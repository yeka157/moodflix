- `components/layout/feature-card-grid.tsx` - Added Tv import from lucide-react, added Tv to iconMap

## Decisions Made
- TV fetch placed in existing `Promise.all` alongside movies and personalized data — parallel execution is always better than sequential for independent network calls
- `trendingTV` made optional with default `[]` on `HomeMovies` — more resilient than required prop (avoids build-time issues if call site ever omits it)
- No `hrefPrefix` on TV MovieRow — `mediaType="tv"` is the correct routing mechanism since `normalizeTVShow` doesn't set `media_type` on returned Movie objects
- Regional TV row deferred — no `getPopularTVInRegion()` function exists; trending row is sufficient for this phase's goal
