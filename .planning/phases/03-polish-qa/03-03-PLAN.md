  <name>Task 1: Fix 32px touch targets on movie card action buttons</name>
  <files>components/movies/movie-card.tsx</files>
  <action>
    The Bookmark and CircleCheck buttons in movie-card.tsx are `h-8 w-8` (32px) — below the 44px minimum touch target required by WCAG 2.5.5. On desktop (hover devices), these buttons are hidden until hover, so 32px is fine. On touch devices, they are always visible as the primary interaction surface for adding to the library.

    Fix: Add `[@media(hover:none)]:h-11 [@media(hover:none)]:w-11` to both motion.button elements. The icon size (`h-4 w-4`) remains unchanged — the extra space is absorbed by the button's flex centering.

    Find both `motion.button` elements in the action buttons section (Bookmark and CircleCheck). Update their className:

    **Bookmark button** — change:
    ```tsx
    className={cn(
      "flex items-center justify-center h-8 w-8 rounded-full transition-colors duration-200",
      isWantToWatch
        ? "bg-primary text-primary-foreground"
        : "bg-black/60 hover:bg-black/80 text-white",
    )}
    ```
    to:
    ```tsx
    className={cn(
      "flex items-center justify-center h-8 w-8 [@media(hover:none)]:h-11 [@media(hover:none)]:w-11 rounded-full transition-colors duration-200",
      isWantToWatch
        ? "bg-primary text-primary-foreground"
        : "bg-black/60 hover:bg-black/80 text-white",
    )}
    ```

    **CircleCheck button** — apply the same className change:
    ```tsx
    className={cn(
      "flex items-center justify-center h-8 w-8 [@media(hover:none)]:h-11 [@media(hover:none)]:w-11 rounded-full transition-colors duration-200",
      isWatched
        ? "bg-green-600 text-white"
        : "bg-black/60 hover:bg-black/80 text-white",
    )}
    ```

    The `[@media(hover:none)]` Tailwind v4 variant targets devices that have no pointer hover capability (touch screens). This ensures desktop hover UX is unchanged while mobile touch targets meet the 44px minimum.

    No other changes to movie-card.tsx in this task.
  </action>
