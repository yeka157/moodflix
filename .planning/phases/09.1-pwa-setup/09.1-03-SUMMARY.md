## Decisions Made

- Script uses `require('@img/sharp-darwin-arm64')` directly — sharp is already installed as a Next.js image optimization dependency, so no additional package needed
- Film-strip perforations carried over from the `MoodflixIcon` brand component — consistent identity from app UI to home screen icon
- Icon background `#09090b` matches the manifest `theme_color` and app background — seamless dark theming on any OS chrome
- M lettermark proportions: 70% for standard icons (fills visible space on any wallpaper), 60% for maskable (preserves safe zone headroom for circular/squircle crops)
